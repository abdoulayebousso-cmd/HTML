# CV Interactif — Abdoulaye Bousso

## Description
CV web moderne et interactif réalisé en HTML5, CSS3 et JavaScript natif (Vanilla JS), dans le cadre du module Développement Web — Licence 2 Informatique, Université Iba Der Thiam de Thiès.

## Technologies utilisées
- **HTML5** — balises sémantiques, structure claire
- **CSS3** — Flexbox, Grid, animations, responsive design, variables CSS
- **JavaScript** — Vanilla JS, IntersectionObserver, localStorage

## Fonctionnalités
- Mode sombre / clair avec sauvegarde dans `localStorage`
- Menu hamburger responsive pour mobile
- Barres de compétences animées au scroll
- Formulaire de contact avec validation côté client
- Bouton "Retour en haut"
- Smooth scroll entre les sections
- Design responsive (mobile, tablette, desktop)

## Structure du projet
```
mon-cv/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── images/
│   └── photo-profil.png
└── README.md
```

## Comment ouvrir
1. Décompresser le fichier `.zip`
2. Ouvrir `index.html` dans un navigateur (Chrome ou Firefox recommandé)
3. Aucune installation requise
